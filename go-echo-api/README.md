# API - Go
## Introdução
Esse repositório armazena o código-fonte das API em Go para o TCC.

## Pré-requisitos
- Go (versão 1.22.6) 
- Echo (versão 4.12.0)

## Tecnologias
A API é desenvolvido em: Go (Linguagem de programação) - Echo (Framework) - PostgreSQL (Banco de dados) - GORM (ORM)

## Project setup

```bash
$ go mod tidy
```

## Run the project

```bash
$ docker-compose up

$ go run main.go
```
